# Shopping Cart System
A Python-based shopping cart system for an online store selling books, music albums, and software licenses.

### Create Virtual Environment

# Windows
python -m venv venv
venv\Scripts\activate

# Mac/Linux
python3 -m venv venv
source venv/bin/activate


## Usage

### Quick Start - Run Demo

python3 examples\demo.py


Functions listed under demo.py file:
- Create products (books, albums, software)
- Add items to shopping cart
- Calculate total price and weight
- Remove items
- Product recommendation


## Testing

# Test products
python tests\test_products.py

# Test shopping cart
python tests\test_cart.py
